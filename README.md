# playon-badge

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
### Generate Web Component
```
npm run build-wc
```

### Using the playon-badge web Component
```
<playon-badge
  text="Label"
  type="primary | secondary | accent | light | dark | highlight | featured"
  icon="live | cancel | calendar | list | camera"
  truncate="15"
  small | large | xl
  bordered
  squared
/>
```
